## SaveFood — Food Rescue Platform

Every day, the US wastes roughly 80 million pounds of food while 1 in 8 Americans faces 
food insecurity. The gap isn't supply — it's coordination.

SaveFood is a concept mobile app prototype that connects restaurants with left over food, local 
nonprofits, food recipients, and donors into one real-time rescue and redistribution system using crowd sourcing concept.

How App Interacts and what's inside
- 4 role-based flows — Restaurant, Nonprofit, Recipient, and Donor
- 22+ linked screens — food posting, live pickup tracking, map discovery, QR check-in, and donation dashboards
- Internet of Things based smart fridge module — simulated sensor freshness scoring, temperature monitoring, and auto-expiry alerts
- Donation transparency — fund utilization breakdowns and impact metrics per nonprofit

Built in Vanilla HTML, CSS, and JavaScript — single file using Prompt engineering, Glean AI and Claude Code
